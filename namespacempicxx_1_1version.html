Detailed Description

This namespace contains all constants and functions to query information of the current library and MPI (library) versions.

Function Documentation

std::string mpicxx::version::mpi_version ( )
inline

The current version of the used MPI standard.

It's in the form of "mpi_version_major.mpi_version_minor".

This function can be called before mpicxx::init() and after mpicxx::finalize() and is thread safe as required by the MPI standard 3.1.
