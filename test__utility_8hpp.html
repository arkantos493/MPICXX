#define EXPECT_THROW_WHAT (   statement,

Value:
do { \
try { \
statement; \
FAIL() << "Expected " #expected_exception; \
} catch (const expected_exception& e) { \
EXPECT_EQ(std::string_view(e.what()), std::string_view(msg)); \
} catch(...) { \
FAIL() << "Expected " #expected_exception; \
} \
} while(false); \

Defines a macro like Google Test's EXPECT_THROW, but also allows to test for the correct exception what() message.

Parameters
[in]statementthe statement which should throw (a specific exception)
[in]expected_exceptionthe type of the exception which should get thrown
