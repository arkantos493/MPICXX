Detailed Description

Provides a class similar to std::source_location.

Author
Marcel Breyer
Date
2020-06-25

Differences are:

  • The new macro MPICXX_PRETTY_FUNC_NAME__ is defined as __PRETTY_FUNC__ (GCC and clang), __FUNCSIG__ (MSVC) or __func__ (otherwise). This macro can be used as first parameter to the static mpicxx::detail::source_location::current() function to get a better function name.
  • Includes a member-function mpicxx::detail::source_location::rank() which holds the current MPI rank (if a MPI environment is currently active).
  • The mpicxx::detail::source_location::stack_trace() function can be used to print/get the current function call stack.

Macro Definition Documentation

#define MPICXX_PRETTY_FUNC_NAME__   __func__

The MPICXX_PRETTY_FUNC_NAME__ macro is defined as __PRETTY_FUNC__ (GCC and clang), __FUNCSIG__ (MSVC) or __func__ (otherwise).

It can be used as compiler independent way to enable a better function name when used as first parameter to mpicxx::detail::source_location::current().
