single_spawner& mpicxx::single_spawner::set_spawn_info ( info  spawn_info)
inlinenoexcept

Set the info object representing additional information for the runtime system where and how to spawn the processes.

As of MPI standard 3.1 reserved keys are:

key description
host a hostname
arch an architecture name
wdir a name of a directory on a machine on which the spawned processes execute; this directory is made the working directory of the executing processes
path a directory or set of directories where the MPI implementation should look for the executable
file a name of a file in which additional information is specified
soft a set of numbers which are allowed for the number of processes that can be spawned
Note
An implementation is not required to interpret these keys, but if it does interpret the key, it must provide the functionality described.

spawn_result mpicxx::single_spawner::spawn ( )
inline

Spawns a number of MPI processes according to the previously set options.

The returned mpicxx::spawn_result object only contains the intercommunicator.

Example:

// create single_spawner
mpicxx::single_spawner ss("a.out", 4);
// add command line arguments
ss.add_argv("--file", "foo", "--size", 42);
// add additional spawn information
mpicxx::info info;
info["wdir"] = "new/working/dir";
ss.set_spawn_info(info);
// spawn new executables
mpicxx::spawn_result res = ss.spawn();
Returns
the result of the spawn invocation
Precondition
The executable name must not be empty.
All command line arguments must not be empty.
maxprocs must not be less or equal than 0 or greater than the maximum possible number of processes (mpicxx::universe_size()).
root must not be less than 0 and greater or equal than the size of the communicator (set via set_communicator(MPI_Comm) or default MPI_COMM_WORLD).
comm must not be MPI_COMM_NULL.
Precondition assertions
If the executable name is empty.
If any command line argument is empty.
If maxprocs is invalid.
If root isn't a legal root.
If comm is the null communicator (MPI_COMM_NULL).
MPI functions internally called
int MPI_Comm_spawn(const char *command, char *argv[], int maxprocs, MPI_Info info, int root, MPI_Comm comm, MPI_Comm *intercomm, int array_of_errcodes[]); // exactly once
