Detailed Description

This class implements all functions that can be called on the result of mpicxx::single_spawner::spawn_with_errcodes() respectively
mpicxx::multiple_spawner::spawn_with_errcodes().

Same as mpicxx::spawn_result but also contains information about potentially error codes.

int mpicxx::spawn_result_with_errcodes::number_of_spawned_processes ( ) const
inline

Returns the number of spawned processes.

Two possible behaviors:

  1. hard spawn: Either maxprocs processes are spawned (returning maxprocs) or the call to spawn results in an error (returning 0).
  2. soft spawn: The info object may specify an arbitrary set of allowed values for the number of spawned processes. If one of these allowed numbers of processes can be spawned, the call to spawn succeeds (returning ). If it isn't possible to spawn one of the allowed number of processes, the call to spawn results in an error (returning 0).
